# Compatibility

fmusim supports

- FMI versions 2.0 and 3.0,
- Model Exchange and Co-Simulation,
- early return and events in Co-Simulation,
- CSV input and output of all data types,
- directional derivatives (with CVode), and
- serialization of the FMU state.

It is tested against against the [Reference FMUs](https://github.com/modelica/reference-fmus).
